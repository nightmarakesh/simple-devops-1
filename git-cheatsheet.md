# Git Cheatsheet for Simple ToDo App

## Работа с ветками

- Просмотр всех веток (локальных и удалённых):
`git branch -a`

- Создать новую фичу ветку (связать с задачей):
`git checkout -b feature/add-edit-task`

- Переключиться на другую ветку:
`git checkout feature/add-edit-task`

- Удалить локальную ветку после мержа:
`git branch -d feature/add-edit-task`

- Принудительно удалить ветку после мержа: 
`git branch -d feature/add-edit-task`

- Переключиться на другую ветку: 
`git checkout master`

- Отображение рабочего каталога и индекса: 
`git status`

- Объединение изменений из одной ветки в другую:
`git merge feature/add-edit-task`

- Последняя версия текущей ветки или последний коммит совершенный в ней:
`git branch -v`

- Вывод списка веток, которые уже слиты с текущей: 
`git branch --merged`

- Копирование файлов из другой ветки в ту, в которой мы сейчас находимся: 
`git checkout feature/add-edit-task app.js`

---

## Коммиты и синхронизация

- Добавить файлы для коммита:
`git add .`

- Добавить конкретный файл для коммита: 
`git add assets/logo.png`

- Сделать коммит с номером задачи:
`git commit -m "feat(#5): add task editing on double click`

- Загрузка последней версии удаленного репозитория:
`git pull origin master`

- Отправление изменений из локально сервера в удаленный:
`git push origin master`

---

## Управление удалёнными ветками

- Обновить локальный список удалённых веток и удалить ссылки на удалённые ветки, которых нет:
`git fetch --prune`

---

## Дополнительные команды

- Просмотреть историю коммитов:
`git log --oneline`

- Просмотреть историю тегов:
`git tag`
